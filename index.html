サンプルコード

size(480, 120);

int y = 60;

ellipse(80, y, d, d);

説明文

　変数とは名前、データ型、そして値を考えて決めます。名前と値は自分で決め、データ型は扱うデータによって決まったコードが定められています。

　変数を使うことで値を再利用することができます。

　

　　int　　　x　　=　60;

　　↑ 　　　↑　　　↑

　データ型　　名前　　 値
